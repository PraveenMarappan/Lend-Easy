# LendEasy – Credit Decision System

LendEasy is a financial web application that automates loan evaluation, risk assessment, and CAM report generation.

## Features
- Loan Application
- Financial Ratio Analysis
- Credit Decision System
- CAM Report Generation
- EMI Calculator

## Tech Stack
- HTML, CSS, JavaScript
- React (Vite)
- Tailwind CSS
