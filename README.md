# 🚀 LendEasy – Credit Decision System

LendEasy is a full-stack financial web application that automates the loan evaluation process, including financial analysis, risk assessment, and Credit Appraisal Memo (CAM) generation. It simulates real-world banking workflows in a simple and interactive way.

---

## ✨ Features

- 📄 Loan Application System  
- 📊 Financial Ratio Analysis  
- 🧠 Credit Decision Engine (Approve / Conditional / Reject)  
- 📑 Automated CAM Report Generation  
- 💰 EMI Calculator with professional output  

---

## ⚙️ Tech Stack

Frontend:
- HTML
- CSS (Tailwind)
- JavaScript
- React (Vite)

Backend (Optional/Extendable):
- Python (Flask)

---

## 🧠 Core Functionality

- Debt-to-Equity Ratio Calculation  
- Profit Margin Analysis  
- Loan Exposure Evaluation  
- Risk Score Computation  
- Loan Approval Decision  
- Recommended Credit Limit & Interest Rate  

---

## 🔄 Workflow

1. Enter borrower details  
2. Perform financial analysis  
3. Calculate ratios  
4. Generate risk score  
5. Make credit decision  
6. Generate CAM report
